# 📮 Postman Collection

## Files

- `Prio_API.postman_collection.json` — API collection
- `Prio_Local.postman_environment.json` — Local environment
- `Prio_Production.postman_environment.json` — Production environment

## Import

1. Open Postman
2. Click **Import**
3. Select all 3 files
4. Choose "Prio Local" environment
